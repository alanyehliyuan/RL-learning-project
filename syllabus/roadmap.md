# Roadmap

## Overall goal

從 RL 新手，逐步走到能獨立理解並實作經典 RL 方法，最後能閱讀常見 deep RL 架構並做小型應用。

## Learning phases

### Phase 0 — Foundations
- probability basics
- expectation / variance
- linear algebra intuition
- optimization basics
- Python / NumPy / plotting

### Phase 1 — Bandits and Intuition
- action-value estimate
- incremental update
- exploration strategies
- regret intuition

### Phase 2 — MDP and Dynamic Programming
- states / actions / transitions / rewards
- return and discounting
- Bellman expectation equation
- Bellman optimality equation

### Phase 3 — Tabular RL
- Monte Carlo prediction/control
- TD(0)
- SARSA
- Q-learning

### Phase 4 — Approximation and Deep RL
- linear approximation
- policy gradient idea
- DQN high-level understanding
- actor-critic overview

### Phase 5 — Projects
- simple benchmark environments
- ablation mindset
- experiment logging
- writeups and reflection

## How to use this roadmap

- 不求一次衝很快，求每一層都真的懂。
- 每個 phase 至少完成一個 exercise 與一個可跑的 implementation。
- 做完後要能用自己的話解釋「這個方法解決什麼問題、限制在哪裡」。
