# Prerequisites

## Math
- basic probability
- expected value
- variance
- conditional probability intuition
- vectors / matrices basics

## Programming
- Python basics
- NumPy
- matplotlib
- basic debugging and plotting

## ML background
- supervised learning 是什麼
- loss / optimization 的基本概念
- train / test 的概念

## Important note

如果某些前置知識不熟，不代表不能開始學 RL。只是要知道自己卡住時，可能不是 RL 本身，而是數學或程式基礎還沒跟上。
