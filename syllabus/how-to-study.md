# How to Study RL with This Repo

## Principle

不要只看 code，也不要只背公式。要在三種視角之間切換：

1. 直覺
2. 數學
3. 實作

## Recommended loop

### 1. Read
先看 module 的概念文件，理解問題設定與核心名詞。

### 2. Solve
先自己做 exercise，不要急著看 solution。

### 3. Implement
把最小版本實作出來，確認你不是只會看答案。

### 4. Reflect
寫下：
- 我原本誤解了什麼？
- 這個方法的核心更新規則是什麼？
- 它跟上一個方法差在哪？

## Working style recommendation

- 每次只學一個主題
- 每次都要留下自己的筆記
- 每個方法都要有一個最小可執行版本
- 盡量畫圖或做可視化，RL 很吃動態直覺
